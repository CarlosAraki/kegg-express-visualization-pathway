About this project

map05171 — 3D isometric log2 fold change bars

Bar height = |log2FC|. Red = up · Blue = down. Data: silver_pathway_areasv1.csv (96 areas).

This interactive visualization shows differential gene expression from the GEO dataset GSE251849, comparing healthy controls with individuals with long COVID. Pathway context is provided by the KEGG reference map map05171 (Coronavirus disease).

Developed for the course Science and Data Visualization in Healthcare (1s2026), taught by Prof. Dr. André Santanchè and Prof. Dr. Murilo Vieira Geraldo.
